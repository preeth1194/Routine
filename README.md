# Routine

A gamified digital vision board app built with React Native.

## Getting Started

```bash
npm install
npm start
```

Then press `i` for iOS simulator or `a` for Android emulator.

## Project Structure

- **app/(tabs)/** - Five tab screens: Home, Daily Routine, Journal, Insights, Profile
- **constants/theme.ts** - Color palette and typography (Plus Jakarta Sans, Satoshi)
- **assets/fonts/** - Custom fonts; see README for adding Satoshi
